Represents the configuration for the compiler.

This struct is created using either config::default_config() or config::parse_matches(matches: &getopts::Matches). It contains a series of fields that can be used to control the behaviour of the compiler.

Example Usage inside the compiler

let cfg = config::Config::default_config();

if cfg.easter_egg {
    println!("Egg!");
}

fn from_matches(matches: &Matches) -> Config

Returns a Config struct by using getopts::Matches to set the fields.

This method analyses a getopts::Matches for fields provided by zwreec_options().

Example

let args: Vec<String> = std::env::args().collect();
let opts = zwreec::config::zwreec_options(getopts::Options::new());

let matches = match opts.parse(&args[1..]) {
    Ok(m) => m,
    Err(f) => { panic!(f.to_string()) }
};

let cfg = zwreec::config::Config::from_matches(&matches);
