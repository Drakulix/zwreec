Module zwreec::frontend::lexer [−] [src]

Builds a token iterator for twee input.

This is lexical analyser using rustlex to analyse twee input and create a lazy evaluating iterator over lexical tokens.

About Rustlex

From rustlex' readme:

RustLex is a lexical analysers generator, i.e. a program that generate lexical analysers for use in compiler from a description of the language using regular expressions.

The Zwreec development team forked rustlex to enable support for some advanced features like preserving accurate line and column numbers for tokens and fix some limitations of the original project. You can find the fork on Github at Farthen/rustlex.

For more information about Rustlex, visit the original repository by LeoTestard.
