pub enum Operand {
    Var(Variable),
    Const(Constant),
    LargeConst(LargeConstant),
    StringRef(LargeConstant),
    BoolConst(Constant),
}

There are three Operands in Z-Code: Variables, SmallConsts and LargeConsts.

The other Operands are for a better code-readability.

Variants

Var

A variable

Const

A small constant

LargeConst

A large constant

StringRef

A string reference

This is internally stored as a large constant containing the string address

BoolConst

A boolean constant (stored as a small integer constant)
