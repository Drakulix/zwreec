zwreec::backend::zcode

Module zwreec::backend::zcode::zfile [−] [src]

The zfile module contains functionality to create a Z-Code file.

This contains all code related to operand handling, Z-Code instruction generation (except for most op-codes) and all of shared structs and code used by codegen and other modules.
