zwreec::utils

Module zwreec::utils::extensions [−] [src]

Extensions to existing types used in the zwreec compiler

Currently this contains only extensions to the Iterator trait yielding new Iterator types that wrap certain functionality, that is used to filter/construct/lookahead all objects passed and iterated upon in the compiler, making it possible to have most of the compiler chain to be lazy_evaluated and even multi-threaded.
