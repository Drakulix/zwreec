macro_rules! error_panic(
    ($cfg:expr => $($arg:tt)+) => (
        {
            if !$cfg.force {
                error!("{}", $($arg)*);
                panic!("Config is set to panic at any error. Try setting the --force flag to ignore this and other errors.")
            } else {
                warn!("{}", $($arg)*);
            }
        }
    )
);

Report an error

Parses the config and reports an error If the config specifies the force option we continue on our way The calling function is expeced to circumvent the error in some reasonable way

Syntax

error_panic!(cfg => MyError)

Example

let cfg = zwreec::config::Config::default_config();
error_panic!(cfg => ExpressionParserError::OperStackIsEmpty);
// Recovery code

Panics

If the config does not specify the force option a panic is generated
